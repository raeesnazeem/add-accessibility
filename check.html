Check ADA Installation - PMs and QAs

Go to the current website.

Right click and go to view page source.

In page source, use ctrl+f or cmd+f to search.

If you're checking for Basic accessibility, search for:

<script src="https://cdn.userway.org/widget.js" data-account="y0juzG0O0x"></script>
Basic Accessibility

If you're checking for Pro/Complete accessibility, search for:

<script src="https://cdn.userway.org/widget.js" data-account="062WMb6Yf6"></script>
Pro Accessibility
Note:

Some websites may have a different script.

In that case, if we find data-account="062WMb6Yf6" in the page source - it is Pro/Complete Accessibility widget.
If we find data-account="y0juzG0O0x" - it is Basic Accessibility widget.
