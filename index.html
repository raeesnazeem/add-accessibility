Pro Accessibility Setup

Steps for the developer:

  1. Navigate to: Elementor > Editor > Custom Elements > Custom Code
  2. Set Heading to: Accessibility UserWay Complete
  3. Enter this code:
<script src="https://cdn.userway.org/widget.js" data-account="062WMb6Yf6"></script>
Make sure to mark the priority as 10 and place it only at <body> end.

Statement Page Requirements:

For pro/complete ADA compliance, an Accessibility page is a must-have. Add this as a new page:

  • Title: Accessibility Statement
  • Slug: accessibility-statement

Generate Statement Content:

  1. Go to the generator website:
    https://raeesnazeem.github.io/userway-statement-generator/
  2. Fill in your business and URL details accurately.
  3. Click Generate Code and then use the Copy functionality to grab the HTML code.
  4. Paste the copied string into a new HTML widget inside your newly created WordPress page.
  5. Verify the layout and make sure it is completely responsive.
